Privacy Policy

Ultimo aggiornamento: 17 giugno 2026

Photo Grid Planner e un'app Android per pianificare una griglia fotografica Instagram, creare anteprime locali e preparare mosaici. L'app non pubblica contenuti su Instagram e non modifica il profilo reale dell'utente.

Dati trattati

L'app puo trattare localmente questi dati:

  • immagini selezionate dall'utente tramite il Photo Picker Android;
  • ordine locale dei post, placeholder, post nascosti e layout salvati;
  • eventuale access token Instagram autorizzato dall'utente;
  • post Instagram disponibili tramite API ufficiale, come URL media, didascalie e timestamp.

Uso dei dati

I dati vengono usati solo per mostrare anteprime, riordinare layout locali, salvare preferenze sul dispositivo e visualizzare i post Instagram autorizzati dall'utente.

Salvataggio

Le preferenze dell'app sono salvate sul dispositivo tramite DataStore. I ritagli generati dal Grid Cutter vengono salvati nella galleria o nella cartella immagini dell'app, su richiesta dell'utente.

Condivisione

Photo Grid Planner non vende dati personali, non usa tracker pubblicitari e non invia dati a server proprietari. Le chiamate a Instagram avvengono solo quando l'utente configura e avvia il collegamento Instagram.

Eliminazione dati

L'utente puo rimuovere i dati locali cancellando i dati dell'app dalle impostazioni Android oppure usando le funzioni di reset disponibili nell'app. Per dettagli aggiuntivi consulta la pagina Richiesta eliminazione dati.

Contatti

Per richieste relative alla privacy, apri una segnalazione nel repository del progetto: Photo Grid Planner su GitHub.
