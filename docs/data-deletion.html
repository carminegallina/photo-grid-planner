Richiesta eliminazione dati

Ultimo aggiornamento: 17 giugno 2026

Photo Grid Planner non gestisce un backend proprietario. I dati dell'app vengono salvati localmente sul dispositivo dell'utente, salvo le chiamate autorizzate alle API Instagram per visualizzare i post nell'app.

Eliminazione dei dati locali

Per eliminare i dati salvati dall'app:

  • apri le impostazioni Android;
  • vai in App > Photo Grid Planner > Memoria;
  • tocca Cancella dati o Disinstalla l'app;
  • rimuovi manualmente eventuali ritagli salvati nella galleria, se desiderato.

Revoca del collegamento Instagram

Puoi revocare l'accesso da Instagram o Facebook nella sezione app e siti web collegati. Dopo la revoca, l'app non potra piu sincronizzare il profilo finche non effettui un nuovo login.
